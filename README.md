
基于webpack4和vue2的多页应用示例，根据vue-cli修改而来

├─build
├─config
└─src
    ├─api // 访问服务器的api接口
    ├─components // 通用组件
    ├─css // 通用样式
    ├─img // 通用图片
    ├─layout // 多页的骨架
    ├─mixins // 通用的mixin
    ├─pages // 页面
    │  ├─index // 首页
    │  │  └─components // 页面里的组件
    │  └─new_page // 新页面
    └─utils // 工具
